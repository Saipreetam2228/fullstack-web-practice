## HTML Lab Practical

This HTML page is a complete lab practical demonstration showcasing the usage of essential HTML tags and page layout elements.

### Concepts Demonstrated
- HTML document structure
- Text formatting tags
- Lists (ordered, unordered, description)
- Images and hyperlinks
- Tables with captions
- Forms with different input types
- Internal page navigation
- Header and footer sections

This file is intended for beginners learning HTML and for academic lab practice.
